# DaylightCreaking
Minecraft plugin for making the creaking keep producing resin during the day
- i've created this plugin for a community server that i am managing for a group of friends,
since we have the 1 person sleep, keeping it night for farming resin is kinda difficult,
so i've decided on creating this plugin to help with this issue.

It seems the creaking keeps producing resin as long as the creaking heart is active,
so in this plugin i am cancelling each time the creaking heart tries to switch states to inactive.
You still need to get night first time in order for the heart to activate and spawn the creaking, but after that it can keep producing.

Note: this is done for 26.1.2, but i've tested it in 26.2 and seems to run just fine
Note2: i will only be updating this plugin according to the needs on the server that i am using it for, but feel free to modify and update it for whatever you need
